# Students-social-media-behavior-analytics-Dashboard
my latest Power BI dashboard project analyzing student social media behavior and its impact on sleep, mental health, and academic performance.
I recently built an interactive Power BI dashboard to analyze how social media usage impacts students' sleep patterns, mental health, and academic performance.

🔍 Key Insights from the Analysis
• Increased daily social media usage is associated with reduced sleep hours
• Higher addiction scores correlate with declining mental health scores
• Around 65% of students reported that social media affects their academic performance

📈 The dashboard includes:
• Executive overview with key KPIs
• Correlation analysis between usage and sleep patterns
• Segment-level risk profiling by gender and academic level

🛠 Tools Used: Power BI, Data Modeling, DAX, Data Visualization
